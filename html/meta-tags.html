HTML Meta Tags

Στην HTML (HyperText Markup Language), η ετικέτα <meta> χρησιμοποιείται για να παρέχει metadata ή πληροφορίες σχετικά με το έγγραφο HTML.
Τα metadata είναι πληροφορίες που δεν εμφανίζονται απευθείας στην ιστοσελίδα, αλλά παρέχουν στους browser πληροφορίες σχετικά με το περιεχόμενο της σελίδας.

Meta tags Περιγραφή
<meta charset='utf-8'>
<meta property='og:description' content='···'> Περιγραφή
<meta property="og:image" content="http://···"> εικόνα
