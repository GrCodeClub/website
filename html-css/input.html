      <input>

      Το <input> είναι ένα ενσωματωμένο στοιχείο (inline element) στην HTML που χρησιμοποιείται για τη δημιουργία εισόδου δεδομένων από τον χρήστη σε μια ιστοσελίδα. Οι δυνατότητες που έχετε με το <input> εξαρτώνται από τον τύπο του πεδίου εισόδου (input type).

      Radio

      <input type="radio" name="gender" value="1"> 1
      1
      2

      Κουτί επιλογής (Checkbox)

      <input type="checkbox">Select 1
      Select 1
      Select 2
      Select 3

      Κατανομή (Range)

      Επιτρέπει στον χρήστη να επιλέξει μια τιμή από ένα εύρος μέσω ενός μπάρας.

      <input type="range" name="volume" min="0" max="100">

      Κουτί κειμένου (Textarea)

      Δημιουργεί ένα πεδίο κειμένου που επιτρέπει στον χρήστη να εισάγει πολλαπλές γραμμές κειμένου.

      <textarea name="comments" rows="4" cols="50">

      Κείμενο (Text)

      Αυτός είναι ο προεπιλεγμένος τύπος. Ο χρήστης μπορεί να εισαγάγει κείμενο.

      <input type="text" name="username">

      Διακομιστής αναζήτησης (Search)

      Αυτός είναι ο προεπιλεγμένος τύπος. Ο χρήστης μπορεί να εισαγάγει κείμενο.

      <input type="search" name="query">

      Κωδικός (Password)

      <input type="password" name="password">

      Κουμπί υποβολής (Submit)

      Χρησιμοποιείται για να υποβάλετε μια φόρμα στον server.

      <input type="submit" value="Υποβολή">

      Κουμπί επαναφοράς (Reset)

      Χρησιμοποιείται για να επαναφέρετε τις τιμές των πεδίων εισόδου στην αρχική τους κατάσταση.

      <input type="reset" value="Επαναφορά">

      Time

      <input type="time">

      Date

      <input type="date">

      Date Time

      <input type="datetime-local">

      Αριθμοί (Number)

      Επιτρέπει στον χρήστη να εισάγει αριθμητικές τιμές. Μπορείτε να ορίσετε το εύρος τιμών με τα min και max.

      <input type="number" name="quantity" min="1" max="100">

      Color

      <input type="url" name="website">

      URL

      <input type="color">

      Email

      <input type="email">

      Upload File

      <input type="file">
